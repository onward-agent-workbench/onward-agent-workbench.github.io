# Onward Agent Workbench — Landing Page

The official landing page for [Onward Agent Workbench](https://github.com/OPPO-PersonalAI/Onward-Agent-Workbench).

## Development

No build step required. Open `index.html` directly or use any static server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

This site is deployed via GitHub Pages from the `main` branch root.
